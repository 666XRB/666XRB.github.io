清理C盘后hexo命令无法识别？

  • 卸载 Hexo：在终端中输入 npm uninstall -g hexo-cli。

  • 重新安装 Hexo：输入 npm install -g hexo-cli。

  1. 确认 Hexo 是否安装在了正确的位置。如果你不确定 Hexo 的安装路径，可以在终端中输入 which hexo（Linux 和 macOS）或 where hexo（Windows）来查找 Hexo 的可执行文件路径。
  2. 然后安装hexo，在命令行中输入 npm list -g，查看是否能找到 Hexo 的安装位置。如果找到了，记下这个路径。，添加环境变量就OK了
