类比推理

一览

出现成语，优先考虑近反义，二级辨析考虑感情色彩。
2.逻辑关系包括全同、并列、包容、交叉、对应关系，可能考查二级辨析。
3.语法关系通过造句解题。
4.语义关系一般考查 1 个，语法关系考查 0-1 个，剩下的题目都考查逻辑关系

【注意】二级辨析：
1.近反义关系：感情色彩。
2.并列关系：出现的先后顺序。
3.工艺：物理、化学。
4.属性：必然、或然。
5.功能：主要、次要。
6.先后顺序：主体是否一致。
7.因果：
（1）原因：人工/自然。
（2）结果：积极/消极。
8.方式目的：方式对错、目的能否达到

（1）语义关系及其常考的二级辨析
（2）并列关系中矛盾关系与反对关系的区分
（3）包容关系中种属关系与组成关系的区分
（4）交叉关系的辨析及其与并列关系的区分
（5）常考的对应关系及其二级辨析

题型

给出一组相关的词，要求通过观察分析，在备选答案中找出一组与之在逻辑关系上最为贴近或相似的词

  • 重难点：
    1.不靠感觉，靠考点；建立出题人的思维体系
    2.学会对比择优，从不严谨到严谨
  • 常见题型：
    1.两词型：A∶B。直接看 A 和 B 的关系。
    2.三词型：A∶B∶C。可以看 A 和 B、可以看 A 和 C、也可以看 B 和 C，比 两词型题目简单，因为只要选项中任意两者的关系错误即可排除
    3.填空型：A 对于（ ）相当于（ ）对于 B。将选项代入题干中， 看前面的关系与后面的关系是否一致

一、语义关系

（一）近义关系、反义关系

遇到成语，优先考虑语义关系;

常考成语积累

  1. 近义关系
    • 谦虚∶虚心
    • 解题技巧：优先看简单、熟悉的成语，用排除法
  2. 反义关系
    • 慎重∶草率
    • 解题技巧：可通过字面意思猜测
  3. 二级辨析（感情色彩）
    • 褒义、贬义、中性词的区分
    • 倾国倾城、锦上添花、火上浇油、贼眉鼠眼
  4. 注意
    • 常见易错成语要积累 目无全牛(技术高超) 别无长物(穷)
    • 优先看简单、熟悉的
  5. 例题解析

（二）比喻象征义

  1. 比喻义与象征义的区别
    • 不用严格区分，知道是比喻象征义即可
  2. 常考例子
    • 如“月亮∶玉盘”“松鹤∶长寿”
  3. 文化常识积累
    • 常见的三字俗语、成语、别名等
  4. 注意
    • 比喻象征义常考文化常识，和中华传统文化息息相关，如追求女孩时可 以送玫瑰花，“玫瑰”象征“爱情”，需要积累常识
    • 题干是比喻义，选项可以是象征义，题干是象征义，选择比喻义也是可以的
  5. 例题解析

“出头鸟”指的是因为表现自己而被别人关注（如枪打出头鸟），此 时和“领头雁”没有关系，但“出头鸟”和“领头雁”都可以指具有领导才能的 人，故二者是并列关系，

二、逻辑关系

（一）全同关系

  1. 定义
    • 两个概念指同一个事物，可以画等号
  2. 常考例子
    • 如“老鼠∶耗子”“西红柿∶番茄”
  3. 解题技巧
    • 造句子判断（A 就是 B，B 就是 A）
  4. 例题解析

“喀斯特地貌”是水溶性地貌，“丹霞地貌”是彩色的，二者不是同 一种地貌

“九一八事变”发生在沈阳，与张学良有关，也称为沈阳事变、奉天事变

“西安事变”和“双十二事变”与张学良有关，二者是同一件事

（二）并列关系

  1. 定义
    • 两个词是同一级别的并列（就像思维导图一样，有同样的根）
    • 两个词如果功能类似，可以判定为并列关系
  2. 矛盾关系与反对关系的区分
    • 矛盾关系：非此即彼，没有第三种情况（对称：非对称）
    • 反对关系：有第三种情况（红色：白色）
  3. 常考例子
    • 如“亲情∶友情”（都是感情根节点，所以是并列）
    • “T 恤∶冬装”（“冬装”是根据季节命名，还有“夏装、秋装”， “冬装”应该和“夏装”是并列的）
    • “牛油∶植物油”（“植物油”和“动 物油”是并列的，“牛油”属于“动物油”，二者级别不同）
  4. 二级辨析（顺序）
    • 如“戊∶己∶庚”，钠∶镁∶铝
  5. 例题解析

两个词如果功能类似，可以判定为并列关系，题干“马匹”“船 只”“车辆”都有运输的功能，故三者为并列关系

看到的是并列关系，且有顺序！

常见的矛盾关系

生∶死 男∶女 白天∶黑夜 A∶非 A
阴∶阳 阴离子∶阳离子 阴刻∶阳刻
曲∶直 动∶静 静态博弈∶动态博弈
软实力∶硬实力 软件∶硬件
主要矛盾∶次要矛盾 实体经济∶虚拟经济
精神文明∶物质文明 急性中毒∶慢性中毒

（三）包容关系

  1. 种属关系

    • 定义：A 是一种 B
    • 解题技巧：用“是”造句，造句通顺为种属关系

  2. 组成关系

    • 定义：A 是 B 的一个组成部分

    • 解题技巧：用“是”造句，造句不通顺为组成关系

      词性不一样，不是组成关系

  3. 常考例子

    • 如“苹果∶水果”“轮胎∶汽车”

  4. 注意

    • 如果题干是矛盾关系，选项没有矛盾关系，可以选择反对关系，因为都属于 并列关系；同理，题干是反对关系，选项有反对关系就选，没有反对关系可以选 矛盾关系。

  5. 例题解析

“影视导演”包括“网剧导演”，“网剧导演”是“影视导演”的一 种，二者是种属关系

城市和乡村是不同的地点，故“城市规划”和“乡村规划”是并列关 系

“城市户口”和“农村户口”是矛盾关系，因为“城市”和“农村” 是矛盾关系（真题考过），但“城市规划”和“农村规划”不一定是矛盾关系， 不影响做题。

（四）交叉关系⭐⭐⭐

题干特征

题干特征：从不同角度描述同一类事物。

如“女士∶公务员”，“女士”
从性别的角度描述，“公务员”从职业的角度描述，从不同的角度描述人，一般可以考虑交叉关系。

拉开差距的题

  1. 定义

    • 两个词中间有交集

  2. 题干特征

    • 从不同角度描述同一类事物

  3. 判定技巧

    • 造句子：“有的 A 是 B，有的 A 不是 B；有的 B 是 A，有的 B 不是 A”
    • 4 句话，都通顺才是交叉关系。

  4. 常考例子

    • 如“女士∶公务员”“木床∶婚床”

  5. 例题解析

“全身麻醉”和“注射麻醉”都是麻醉，“全身”看的是范围， “注射”看的是方式，从不同角度描述麻醉

造句看成立否

“有的全身麻醉是注射麻醉， 有的全身麻醉不是注射麻醉（呼吸麻醉）；有的注射麻醉是全身麻醉，有的注射 20 麻醉不是全身麻醉（局部麻醉）”

所有的“加热消毒”都是“物理消毒”，为种属关系

“抽样调查”是根据范围划分，“问卷调查”是根据调查形式划分（可 以根据问卷、电话、座谈会进行调查），从不同角度描述调查，造句为“有的抽 样调查是问卷调查，有的抽样调查不是问卷调查；有的问卷调查是抽样调查，有 的问卷调查不是抽样调查”，二者是交叉关系

历年考法

团扇∶羽毛扇∶舞蹈扇（2018 联考） 旱田作物∶高产作物（2018 江苏）
圈椅∶实木椅∶办公椅（2018 联考） 哺乳动物∶陆生动物（2016 山东）
推理漫画∶连环漫画（2018 吉林） 内陆湖∶淡水湖（2019 国考）
馆藏文物∶史前文物（2018 吉林） 教师∶陕西人（2017 陕西）
川菜∶素菜∶热菜（2017 天津） 诗人∶教授（2017 江苏）
壁画∶山水画（2015 吉林） 专家∶官员（2015 江苏）

（1）团扇∶羽毛扇∶舞蹈扇：都在说扇子，“团扇”根据形状划分，“羽毛扇”根据原材料划分，“舞蹈扇”根据功能划分，三者是从不同角度划分的，
为交叉关系。
（2）圈椅∶实木椅∶办公椅：“圈椅”根据形状划分，“实木椅”根据原材料划分，“办公椅”根据功能划分，从不同角度划分。
（3）推理漫画∶连环漫画：“推理漫画”根据内容是否涉及推理划分，“连环漫画”根据形式划分，二者是从不同角度划分的，为交叉关系。
（4）馆藏文物∶史前文物：“馆藏文物”根据是否有收藏划分，“史前文物”根据时间划分，从不同角度᧿述文物，为交叉关系。
（5）川菜∶素菜∶热菜：“川菜”根据地点划分，“素菜”根据是否有肉划分；“热菜”根据是凉拌菜还是热菜划分。
（6）壁画∶山水画：“壁画”根据画的地点划分，“山水画”根据画的内容划分，为交叉关系。
（7）旱田作物∶高产作物：“旱田作物”根据是否有水划分，“高产作物”根据产量高低划分，从不同角度划分。
（8）哺乳动物∶陆生动物：“哺乳动物”根据是否胎生哺乳划分，“陆生动物”根据生活地点划分。
（9）内陆湖∶淡水湖：“内陆湖”根据位置在里面分，“淡水湖”根据盐多少划分。

涉及人的身份、职业，优先考虑交叉关系，因为一个人可能有多个身份， 一个人既可以是老师，又可以是陕西人；既可以是诗人，也可以是教授；既可以 是专家，也可以是官员。

（五）对应关系

  1. 原材料和工艺（成品 ）

    • 葡萄∶酿造∶葡萄酒

    • 常考二级辨析：物理变化、化学变化

    • 涉及物理变化的工艺：印刷、捆扎、打磨、雕琢、剪裁、织造等
      涉及化学变化的工艺：燃烧、氧化、发酵、酿造、冶炼等
      区分标准：有无新物质生成

  2. 功能

    • 灯∶照明∶装饰（一般为名词+动词）

    • 常考二级辨析：主要功能、次要功能

  3. 属性

    • 盐∶咸 花∶红（一般为名词+形容词）

      属性一般对内，或者带XX性的就是属性、

    • 常考二级辨析：必然属性、或然属性

    • “潮汐能”和“生物质能”是两种不同的能源，二者为并列关系， “可再生”是对内的，为属性

  4. 时间顺序

    • 先考虑时间先后顺序，如果选不出答案，再考虑主体是否一致

    • 恋爱∶结婚∶生子（一般为动词+动词）

    • 常考二级辨析：主体是否一致

    • “铁路铺设”和“桥梁建造”没有先后顺序

      2019 年国考考过真题， 因为有民事诉讼，和刑事诉讼，在民事诉讼中可以先“提起公诉”，再“立案审 查”，如果涉及刑事诉讼，比如杀人了，应该先“立案审查”，再“提起公诉”。 题干中“公诉”的一般是刑事案件，所以“立案审查”和“提起公诉”有先后顺 序，但是应该是先“立案审查”，再“证据确凿”，最后“提起公诉”，比如《人 民的名义》中侯亮平先找证据再去公诉

  5. 因果关系

    • 因果关系也有先后顺序，先因后果，区分时间和因果可以造句，可以用“导 致”造句，则为因果关系，如果不能用“导致”造句，则为先后顺序。

    • 常考二级辨析：原因（自然/人工）、结果（积极/消极）

    • 必然因果/或然因果：可以考查，但考的概率很低，因为必然因果很少

  6. 方式目的

    • 因果关系和方式目的区分：一般用造句子区分，能用“导致”造句是因果 关系，能用“为了”造句是方式目的

    • “刻舟∶求剑”，“刻舟”是为了“求剑”，“刻舟”是方 式，“求剑”是目的。

    • 常考二级辨析：方式对错、目的能否达到

    • .国家的政策喜欢考查方式目的，涉及国家的政策，优先考虑方式目的

  7. 例题解析

三、语法关系

  1. 造句子技巧

    • 主谓关系、动宾关系、主宾关系、偏正关系

  2. 区分动词和动宾

    • 动宾是有对象的，没有对象的是动词

  3. 偏正关系

    • 形容词修饰名词、副词修饰动词、名词修饰名词

      如“美丽∶花朵”，也可以说“美丽”是“花朵”的属性，中 间有“的”造句就是偏正。最简单的偏正是形容词修饰名词，也叫属性；第二种 是一个副词修饰动词，比如“慢慢地跑”，“跑”是动词，“慢慢”是副词，中 间用“地”连接；第三种是名词+名词，如“历史老师”，“历史”和“老师” 都是名词，但是“历史”修饰“老师”，为偏正关系。

  4. 造句子原则：尽量不加字，加字尽量加一样的

  5. 例题解析

四、学习建议

  1. 建立出题人思维体系
    • 不靠感觉，靠考点解题
  2. 对比择优
    • 从不严谨到严谨，考虑二级辨析
  3. 刷题提升
    • 类比推理需要大量刷题才能提高正确率

成语积累

积累成语：
（1）一望无垠：宽广、辽阔，一眼望去看不到边际。
（2）人云亦云：指没有主见，随声附和别人。
（3）耳提面命：形容恳切地教诲。
（4）方兴未艾：事物正在兴起、发展，一时不会终止。
（5）厉兵秣马：形容做好战前准备。
2.“耳提面命”和“春风化雨”是近义关系，“春风化雨”指受到好的熏陶、教育

（1）过河拆桥：比喻达到目的以后，就把曾经帮助过自己的人一脚踢开。
（2）饮水思源：比喻人在幸福的时候不忘幸福的来源。
（3）江心补漏：比喻补救已迟，无法挽回差错或败局。
（4）未雨绸缪：比喻事先做好准备。
（5）不名一钱：指没有一文钱，形容极其贫困。
（6）一贫如洗：形容十分贫穷。
（7）了如指掌：形容对情况非常清楚。
（8）一窍不通：比喻一点儿也不懂。
（9）浮光掠影：形容印象不深刻。
（10）走马观花：比喻粗略地观察事物。

三字

【注意】成语积累：
1.领头雁：比喻在众人眼里有一定的号召力和领导能力，具有榜样力量的。
2.拓荒牛：比喻开拓创新的先行者。
3.出头鸟：比喻表现突出或领头的人，也比喻因出头露面而招人注目的人。
4.先行者：指首先倡导的人。
三字俗语类积累
孺子牛：比喻心甘情愿为人民大众服务，无私奉献的人
烂摊子：比喻秩序混乱，不易收拾整顿的局面或单位
白眼狼：比喻忘恩负义、过河拆桥、恩将仇报的人
墙头草：比喻没有主见、左右摇摆、见风使舵的人
纸老虎：比喻貌似强大、实际虚弱的人或集团
及时雨：比喻在关键时刻能解救危难的人或事
冷板凳：比喻因不受重视而担任清闲的职务
捞稻草：比喻在绝望中做徒劳无益的挣扎
跑龙套：比喻在人手下做无关紧要的事
定心丸：比喻能使人情绪安定的东西
耍花枪：比喻卖弄小聪明的欺骗行为
拉清单：比喻结算彼此的恩怨利害

比喻象征义

如“胃口∶兴趣”，“胃口”比喻“兴趣”。
2.如“手足∶兄弟”，手足如兄弟，“手足”比喻“兄弟”。
3.如“总角/垂髫∶小孩”，“总角/垂髫”比喻“小孩”。
4.如“花甲/耄耋/伛偻/黄发∶老人”，“花甲/耄耋/伛偻/黄发”比喻“老
人”。
5.如“丝竹∶音乐”，无丝竹之乱耳，无案牍之劳形，“丝竹”比喻“音乐”。
6.如“汗青∶史册”，留取丹心照汗青，“汗青”比喻“史册”。
7.如“青衿∶读书人”，青青子衿，悠悠我心，“青衿”比喻“读书人”。
8.如“白丁/布衣∶百姓”，“白丁、布衣”比喻“百姓”。
