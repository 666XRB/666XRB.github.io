逻辑论证1（削弱）

一、基础概念

（一）论证的定义与特点

论证是一种逻辑推理过程，通过论据和推理方式来支持或反驳某个观点。其核心在于论点、论据和论证关系的有机结合，形成有说服力的推理。

论证和言语题截然不用，言语、日常结论、组合排列、

翻译推理都是由一段话 可以推出选项的哪个结果

而论证题是给出一段话，需要用选项的内容证明这段 话是否成立。

所以整个逻辑过程是和日常做题相反的，加强可能可以靠语感做题， 但削弱靠语感做题非常难。

（二）论证的构成要素

找到论点，论据一般在论点附近（前/后），根据就近原则，在附近找

  1. 论点：是作者明确提出的观点或结论，是论证的核心。它像一面旗帜，展示作者的立场。例如，“必须采取行动保护环境，以确保地球的可持续发展”就是一个论点。

    论点提示词包括“所以，结论是，这表明/说明/意味着，由此推出，据此认为/推论”等，

    这些词后面通常跟着结论性的话。在题目中，先找这些提示词，90%以上的论证题目都有这些词，对这些词要敏感。

  2. 论据：是用来支持论点的理由、事实、数据、例子等。它是论点的支撑，像建筑物的基石。例如，如果论点是“锻炼对健康有益”，那么“研究表明，每周进行至少150分钟的中等强度锻炼可以降低患心脏病的风险”就是一个论据。

    论据提示词有“由于、因为、鉴于、根据、理由是”，后面跟的是论据，还包括原因、数据、事例、实验或调查内容等。

    判定一句话是否是论据，看它能否解释论点，因为论据的作用就是解释论点。

  3. 论证关系（桥）：是连接论点和论据的逻辑纽带，使论据能够合理地支持论点。就像桥梁连接两岸，论证关系将论据和论点紧密相连，使整个论证结构完整、逻辑严密。例如，论据“吸烟会损害肺部功能”和论点“吸烟对健康有害”之间的论证关系就是因果关系。

（三）首尾句原则

遇到没有标志词的题目，遵循首尾句原则，往往在文段的第一句或最后一句中找论点。

这和言语理解题类似，“总-分-总”“分-总”“总-分”的结构中，第一句和最后一句比较容易是整个文段的总结性表述。一道题目可能有多个论点，如有“研究人员认为”和“专家据此推论”，看起来“专家据此推论”更像是论点，但若问的是“最能加强研究人员研究结果的是”，此时要以研究人员的话当作论点。结合提问方式，问啥找啥。

二、解题步骤

（一）明确提问方式

首先要明确题目是削弱题还是加强题，或者是其他类型的逻辑推理题。不同的题型有不同的解题思路和方法。

  1. 削弱类（只要和题干反着来，让在题干中挑毛病的 题都是削弱类的）
  • 常见提问方式：
    • 如果以下各项为真，最能质疑/反驳/削弱/反对上述论断的是？或者证明。。错误，挑战
    • 如果以下各项为真，（最）不能质疑上述论断的是？

（二）分析题干

  1. 找出论点：通过论点提示词或者首尾句原则，快速定位题干中的论点。论点是整个论证的核心，准确找到论点是解题的关键。
  2. 找出论据：通过论据提示词或者分析论点的支撑内容，找到题干中的论据。论据是用来支持论点的理由、事实、数据、例子等。

（三）预设削弱方向

根据论点和论据的关系，预设可能的削弱方向：

  1. 论点与论据话题一致：如果论点和论据讨论的是同一个话题，优先考虑否论点。即直接否定论点的表述，这是最直接、最有力的削弱方式。
  2. 论点与论据话题不一致：如果论点和论据讨论的话题不一致，考虑拆桥。即说明论据不能直接支持论点，切断论点和论据之间的逻辑联系。
  3. 特殊题型：
    • 双方互怼：题干中有两个对立的观点，此时考虑否论据。即否定对方论据的真实性或有效性，从而削弱对方的观点。
    • 因果论证：论点中包含因果关系，此时考虑因果倒置或他因削弱。因果倒置是将因果关系颠倒，他因削弱是引入其他可能的原因，说明论点中的因果关系不成立。

（四）匹配选项与择优

根据预设的削弱方向，逐一分析选项，找出最符合的选项。在多个选项都能削弱的情况下，进行力度比较，选择削弱力度最大的选项。

三、常见削弱方式

（一）否论点

》绝大多数的削弱题都考查否论点

  1. 选项特征：选项的表述与论点直接相反，直接否定论点的正确性。

  2. 适用情况：文段只有论点、无论据 或者文段的论点和论据话题一致

    如何判断论据和论点讨论的话题一致?

  3. 力度分析：否论点是最直接、最有力的削弱方式，因为它直接否定了题干的核心观点，使整个论证失去立足点。

  4. 注意：

    1. 正确答案无论是加强还是削弱，一般都会提到论点的关键词

    2. 解题步骤：文段只有论点/文段的论点和论据话题一致，找出论点→预设 答案（直接反着说）→对比选项。

    3. 直接否和举例否都可以削弱。

      举例否需要多看一部。能不能让论点不成立。例如上面的两种情况。

    4. 力度择优原则：整体＞部分 C选项

    5. 只有论点的题目

    6. 注意臆想以及工作效率+论点的两句话否定其中一句是可以的+择优原则——整体＞部分

小结

【注意】削弱论点小结（力度最强&最常考）：
1.只有论点或论点、论据话题一致：优先考虑否论点。
2.正确答案往往与论点中的关键词一致或同义替换。
3.论点较长或无法找到论点：对文段进行概括（概括出一个论点）。
4.注意力度的比较：整体＞部分；必然＞可能（说得越绝对越好，加强/削弱有力度要求）；与论点越一致越好。

（二）拆桥

  1. 定义与核心：拆桥是切断论点和论据之间的逻辑联系，说明论据不能有效支持论点。

  2. 适用情况：当没有否论点的选项，或者论点与论据话题不一致！设问方式为削弱“论证”（不一定）

    当论点和论据之间存在逻辑漏洞，或者论据不能充分支持论点时，可以使用拆桥的方式进行削弱。

  3. 表述形式：拆桥的选项通常会说明论据与论点无关、论据不能推出论点等。

  4. 注意

    1. 拆桥有多种表达形式，只要能让论据不等于论点，就是拆桥。

      论点：我很幸福

      论据：我很有钱

      A.有没有钱和幸不幸福没有必然关系

      B.有钱的人也会不幸福

      C.有钱不是衡量幸福的唯一指标

    2. 论点为整体，论据为部分，有两种拆桥形式。 1.拆桥 ：部分代表不了整体。 2.拆桥 2：其他部分不这样。

    3. 整体大于部分

    4. “时髦”拆桥

      论据：某物质中的一些元素有效

      论点：某物质有效

      削弱：某物质内该元素含量极少/要吃极多

      论据：苦瓜水提取物，可降低Ⅱ型糖尿病小鼠的高血糖。 论点：Ⅱ型糖尿病患者多吃苦瓜有助于降低血糖水平。

      B.日常食用的苦瓜中，苦瓜水提取物含量极少

    5. 拆桥的不同方式

      今天特别热，因为今天出太阳了
      ①出太阳推不出天气热
      ②出不出太阳跟热不热没关系
      ③有时候出太阳也不热
      ④出太阳不是影响气温升高的唯一因素

    6. 若考某物质和某元素整体与部分的关系，只考这个元素和部分的关 系，其他元素太多了。

（三）否论据

  1. 特征：选项的表述与论据直接相反，否定论据的真实性或有效性。

  2. 适用情况：题干中有“双方互怼”，经常采用否定论据来削弱 ；题干有论据且在找不到否论点和拆桥

  3. 题型标志：题干中有明显的论据，且论据对论点的支持作用较大。

  4. 注意

    1. 区分上面

    2. 双方互怼->否论据

（四）因果倒置

因果倒置：时间先后顺序颠倒，不用看语序是否颠倒。判断选项是否为因 果倒置时，就看选项中同时出现的因和果的时间先后顺序与题干是否一致，若时 间先后顺序颠倒了，则为因果倒置。

  1. 定义与特征：因果倒置是将论点中的因果关系颠倒，说明结果可能是原因，原因可能是结果。

  2. 适用情况：当论点中明确包含因果关系，且存在可能的因果倒置情况时，可以使用因果倒置的方式进行削弱。

    ……是……的原因；
    ……导致/使得/有助于……；
    ……增加/降低/加强/减轻……；
    这是由于……

  3. 易错：

    论点：1 导致 2
    因果倒置削弱：2 导致 1
    落后（先）导致挨打（后） 因果倒置——（先）挨打使得（后）落后
    喝可乐（先）让我胖了（后） 因果倒置——（先）胖子才喝可乐（后）

    但是不能说用前面的翻译，A->B这跟翻译是不一样的！！

    因果倒置选项特征：因果同时出现且时间顺序颠倒

  4. 注意

    1. 例子

    2. 例题（因果倒置特征——因果同时出现且时间顺序颠倒。）

（五）他因削弱

  1. 定义与本质：他因削弱是引入其他可能的原因，说明论点中的因果关系不成立，或者存在其他影响因素。

    承认结果，同一个主体、同时存在、两种及以上可能的原因 若论点是其中一种原因，则可用另一种原因进行削弱

    他因的本质：

    干扰因素——用于削弱论点原因与结果之间的唯一关系

    他因的特点：同一主体、同一时间、不同原因

  2. 适用情况：当论点中包含因果关系，且存在其他可能的原因时，可以使用他因削弱的方式进行削弱。

  3. 判断方法：选项中会引入与论点中不同的原因，且这个原因可能对结果产生影响。

  4. 注意

    1. 例题

    2. 例题

      “发挥一定的防御作用”这种太含糊的话

    3. 当选项主体和论点的主体是并列或者交叉关系，那么这个选项就是无关！

      “题干讨论“该市”，但选项讨论“临近省市”，主体不一致，无关项， 保留。”

解题时，需要排除削弱项，选择加强项或无关项。特殊提问
不用预设，选项可能有多种加强/削弱方法，结合排除思维解题

不能削弱——排除削弱选项，加强与无关均可

五、注意事项

（一）论点的重要性

论点是整个论证的核心，解题时必须准确找到论点。论点通常通过提示词或者首尾句原则来定位。

（二）话题一致性的判断

判断论点和论据的话题是否一致是解题的关键。话题一致时，优先考虑否论点；话题不一致时，考虑拆桥。

（三）力度比较原则

在多个选项都能削弱的情况下，需要进行力度比较。一般来说，否论点的力度最大，其次是拆桥、否论据、因果倒置和他因削弱。

六、削弱解题思路总结

判断论据和论点讨论的话题一致

在逻辑推理题中，判断论据和论点讨论的话题是否一致是解题的关键步骤之一。

1. 分析论点和论据的主体

（1）论点和论据的主体对象是否一致

首先，要明确论点和论据分别讨论的是什么主体对象。主体对象可以是人、物、事件、概念等。如果论点和论据的主体对象不同，那么它们讨论的话题就不一致。

例1：

论点：甲公司今年的销售额增长了20%。

论据：乙公司的市场份额在今年有所下降。

两者的主体对象不同（甲公司与乙公司），因此论据和论点讨论的话题不一致。

（2）主体对象不一致再看是否具有相关性

即使论点和论据的主体对象不同，也需要判断它们是否具有相关性。如果主体对象之间存在某种联系，那么论据可能仍然可以支持论点。

例2：

论点：甲公司的销售额增长得益于其新产品的推出。

论据：甲公司的新产品在市场上获得了高度评价。

在这个例子中，论点讨论的是甲公司销售额增长的原因，而论据讨论的是甲公司新产品的市场评价。虽然论据没有直接提到销售额，但它与新产品相关，而新产品与销售额增长有关联，因此论据和论点讨论的话题具有一致性。

2. 比较论点和论据的核心概念

（1）识别论点和论据中的核心概念

核心概念是论点和论据中表达主要观点或事实的关键词汇。这些核心概念通常包括名词、动词、形容词等。

例3：

论点：锻炼对身体健康有益。

论据：研究表明，定期进行有氧运动可以降低患心脏病的风险。

在这个例子中，论点的核心概念是“锻炼”和“身体健康”，而论据的核心概念是“有氧运动”和“患心脏病的风险”。可以看出，有氧运动是锻炼的一种形式，降低患心脏病的风险与身体健康相关，因此论据和论点讨论的话题一致。

（2）判断核心概念是否一致或相关

如果论点和论据中的核心概念相同或具有相关性，那么它们讨论的话题可能一致。如果核心概念完全不同，那么论据和论点讨论的话题可能不一致。

例4：

论点：阅读文学作品可以提高人的审美能力。

论据：某学校增加了体育课程的课时。

在这个例子中，论点的核心概念是“阅读文学作品”和“审美能力”，而论据的核心概念是“学校”和“体育课程”。两者的核心概念完全不同，没有相关性，因此论据和论点讨论的话题不一致。

3. 分析论点和论据的逻辑关系

（1）确定论据是否直接支持论点

论据的目的是支持论点，因此需要判断论据是否直接为论点提供证据或理由。如果论据能够直接证明论点的正确性，那么它们讨论的话题一致。

例5：

论点：吸烟对健康有害。

论据：吸烟会导致肺癌等多种疾病。

在这个例子中，论据直接说明了吸烟会导致疾病，从而支持了吸烟对健康有害的论点，因此论据和论点讨论的话题一致。

（2）判断论据是否与论点存在间接联系

有时，论据可能并不直接支持论点，但通过某种逻辑关系与论点相关联。这种情况下，需要判断这种间接联系是否合理。

例6：

论点：城市应该增加更多的公园和绿地。

论据：研究表明，绿色空间可以改善城市居民的心理健康。

在这个例子中，论据并没有直接说明增加公园和绿地的具体好处，但它通过绿色空间与心理健康的关系间接支持了论点。因此，论据和论点讨论的话题具有一致性。

4. 检查论据是否偏离主题

（1）判断论据是否引入无关信息

如果论据引入了与论点无关的信息，那么它可能偏离了主题，导致论据和论点讨论的话题不一致。

例7：

论点：应该加强学校的食品安全管理。

论据：某学校最近安装了新的监控设备。

在这个例子中，论据提到的是学校安装了新的监控设备，这与食品安全管理没有直接关系，属于无关信息，因此论据和论点讨论的话题不一致。

（2）评估论据的相关性

即使论据中包含了一些与论点相关的元素，也需要评估其整体相关性。如果论据的主要内容与论点无关，那么它仍然可能偏离主题。

例8：

论点：在线教育可以提高学习效率。

论据：某在线教育平台提供了丰富的课程资源，并且拥有优秀的教师团队。

在这个例子中，论据提到的课程资源和教师团队是在线教育的一部分，但它们并没有直接说明如何提高学习效率。因此，论据与论点的相关性较弱，讨论的话题不完全一致。

通过以上方法，可以较为全面地判断论据和论点讨论的话题是否一致。在实际做题过程中，需要灵活运用这些方法，结合具体题目进行分析。

力度比较

如果出现因果倒置的时候，出现例如

1倒置2

但是给了2倒置3.那么也是无关选项

一般情况下，出现因果倒置的选项，优选

他因削弱：针对论点进行否定，也属于否论点，但为可能性削弱，力 度弱于因果倒置、直接否论点。

3.直接否论点与因果倒置力度一样，他因、举反例力度较弱（他因、举反例 无法进行力度比较，都属于或然性削弱，不属于必然性削弱）。

课上提到的力度 比较会考，没提到的力度比较在逻辑中属于不同维度，无法进行比较，不用纠结。

思路

1.只有论点，直接否论点。
2.有论点、论据，话题一致，考虑否论点；话题不一致，考虑拆桥。
3.出现“双方互怼”的特殊题型，考虑否论据。
4.如果论点有因果关系，考虑因果倒置、他因。削弱题中考试真正能遇到的力度比较就是“因果倒置＞他因”，其他的力度比较基本不考，若考，最多会考“否论点＞否论据”，不怎么考否论点、拆桥的力度比较。

  1. 先看提问明确题型：确定是削弱题还是其他类型的逻辑推理题。
  2. 找出题干中的论点和论据：通过提示词或者首尾句原则，准确找到论点和论据。
  3. 判断论点和论据的话题是否一致：
    • 如果一致，优先考虑否论点。
    • 如果不一致，考虑拆桥。
  4. 遇到双方互怼的题型，考虑否论据：否定对方的论据来削弱对方的观点。
  5. 如果论点包含因果关系，考虑因果倒置或他因削弱：颠倒因果关系或者引入其他原因来削弱。
  6. 根据预设方向匹配选项，并进行力度比较择优：选择最符合的选项，并在多个选项中选择削弱力度最大的。

一览

复习的重点是否论点、因果类论点。
2.答疑：
（1）他因起削弱的作用，原因解释起加强的作用。
（2）无关项既不能削弱，也不能加强
